When I ran the query multiple times, I noticed it sometimes right, sometimes wrong (failed test). This is a problem because LLM can generate different response each time if we set the tempature for LLM to not be 1. For CI/CD to work reliably, we need mock LLM mode.

Mock LLM mode for CI testing

Mock LLM mode returns the retrieved text directly, which makes tests deterministically fail. Without mock mode, tests would not be predictable. For automated CI, we need predictable results.
